### Google Analytics

1. Ask a GCP admin to create a new `property` for you in Google Analytics under `Aptos Labs`. Provide the following
  1. Website name (e.g., docs.aptos.dev)
2. Once the `property` is created, ask for the `Measurement ID`

#### GCP Admins

> Ignore the steps below unless you have Admin access

1. Navigate to [analytics.google.com](analytics.google.com)
2. In Google Analytics Admin (`Admin` button is the gear icon in the bottom left of the screen), click Create, then select `Property`.
3. Provide a property name.
4. For the section on `Choose your business objectives`, select `Generate leads` and `Examine user behavior`
5. For the section on `Start collecting data`, choose `Web` as the platform
6. Setup the data stream

### Installation

> Note: Make sure you have installed `node >= 20`, and `pnpm` before beginning

To create a new docs site from scratch, navigate to your repo or the shared docs repository and do:

```sh filename="Terminal"
pnpx @aptos-labs/create-docs
```

This should prompt you to enter the name of the docs site and will configure a brand new Aptos Docs site with Nextra.

```sh filename="Terminal"
pnpm install
```

### Develop
